Drawing State

Sub VoxSetVolumeType(T As VoxVolumeType)
Changes the volume type of the current volume. (without changing its content) 

Sub VoxSetColor(C As UInteger)
Changes the current color used by drawing commands.

Colors are 32 bit numbers compatible with FreeBASIC's RGB() and RGBA() macros. Please note that while 8 bit alpha can be stored in a voxel, only discrete (1 bit alpha) transparency can actually be rendered. Any non-zero alpha value will render as opaque. As such 0 can be specified for transparent voxels and RGB() can be used to indicate the color of all opaque voxels. 

Sub VoxSetVolume(V As Vox_Volume)
Changes the current volume to the one referenced by V. All subsequent drawing operations will draw on to this volume. 

Sub VoxSetSource(V As Vox_Volume)
Changes the source volume to the one referenced by V. The source volume is the source of voxels used by the VoxBlit function. It is planned that future versions will use the source volume as a texture for drawing commands supporting texturing.

Blit State

Sub VoxSetBlitDefault
Resets the Blit transformation to the identity. 

Sub VoxBlitRightRotate(Axis As UInteger, ByVal Amount As Integer = 1)
Creates a new Blit transformation by appending an Amount number of 90 degree rotations about Axis. 

Sub VoxBlitReflect(Axis As UInteger)
Creates a new Blit transformation by appending a reflection about Axis to the current Blit transformation.

Axis may be one of the following: 
VOXEL_AXIS_X
VOXEL_AXIS_Y
VOXEL_AXIS_Z
