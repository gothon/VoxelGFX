Voxel Vectors

Type Vec3I
    As Integer X, Y, Z
End Type
This is a basic vector type for storing voxel coordinates in integers. Many VoxelGFX functions take this type as a parameter and most common vector arithmetic operations are implemented into the type.
For example: 
...
VSet Vec3I(1, 2, 3)
Var A = Vec3I(1, 1, 1)
VoxLine A, 10*A
...

Volume Types

Enum VoxVolumeType
    Volume_Dynamic = 0
    Volume_Static = 1
    Volume_OffScreen = 2
End Enum
Enumeration for volume types.
- Dynamic volumes are used for drawing and rendering and maintain both OpenGl and system copies for fast updating.
- Static volumes are optimized for rendering and use only OpenGl managed memory. Static volumes perform extra hidden surface removal whenever they are changed thus slowing drawing but speeding rendering operations.
- Off Screen volumes are optimized for drawing operations and use system memory only. 

Type Vox_Volume As Integer
This type is used to reference volumes.
